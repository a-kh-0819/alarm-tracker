# アラームトラッカー

このアプリはAndroid向けのシンプルなFlutter製目覚ましです。アラーム機能に加え、起床した時刻を記録して一覧表示します。デザインはモダンでシンプルなレイアウトを採用しています。

## 機能
- 指定した時刻にローカル通知でアラームを鳴らします。
- アラームを止めた時刻を記録し、`shared_preferences`に保存します。
- 記録した起床時間をリストで確認できます。

## 始め方
1. Flutterをインストールし、Android開発環境を準備します。
2. `flutter pub get` を実行して依存関係を取得します。
3. `flutter run` で端末またはエミュレーター上でアプリを起動します。
